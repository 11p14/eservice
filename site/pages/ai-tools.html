Photo Enhancer, Remove Background — এই কাজগুলো ভারী ইমেজ প্রসেসিং, যেটা Google Apps Script দিয়ে করা যায় না। এর জন্য একটা ফ্রি-টায়ার থার্ড-পার্টি API লাগবে, যেমন remove.bg (ব্যাকগ্রাউন্ড রিমুভ) বা TinyPNG/Deep-Image (এনহান্সমেন্ট)। ধাপ: (১) ওই সার্ভিসে ফ্রি API key নেবে (২) ব্রাউজার থেকে ইউজারের ছবি সেই API-তে পাঠাবে (৩) রেজাল্ট ইমেজ ফেরত এসে পেজে দেখাবে। Passport Size Maker সম্পূর্ণ ক্লায়েন্ট-সাইড Canvas API দিয়েই বানানো সম্ভব (কোনো API লাগবে না)।
